simple minecraft skin viewing util, *probably* not production ready, nor stable

**features:**
- supports loading skins from minecraft servers, url and local png file
- automatic slim and classic skin detection
- pre-1.8 skin conversion to 1.8
- free camera movement
- toggling on and off body parts
- reload button

not an official minecraft project, im not associated with mojang nor microsoft, 2 skins in assets dir copyright mojang
